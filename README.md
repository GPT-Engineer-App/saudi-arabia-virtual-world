## Setup

```sh
git clone https://github.com/GPT-Engineer-App/saudi-arabia-virtual-world.git
cd saudi-arabia-virtual-world
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
